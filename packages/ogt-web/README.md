# ogt-web

Web-based grid editor for [openGrid](https://github.com/ddanier).

- Click tiles to toggle them on/off (create holes)
- Toggle features (connectors, chamfers, screws) at individual summits
- Adjust screw parameters (diameter, head diameter, head inset)
- Export as JSON, compact code, or a ready-to-paste `uvx ogt generate` command

## Development

```bash
npm install
npm run dev
```
